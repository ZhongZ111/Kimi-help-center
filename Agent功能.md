深度研究

1.深度研究是什么
Kimi 的深度研究（Deep Research）是 Moonshot AI 推出的一项 AI 驱动的自主调研功能，本质是一个多步骤 AI Agent 工作流。
<img width="2996" height="1494" alt="image" src="https://github.com/user-attachments/assets/0ebbfd3d-7487-421e-975b-c3ab61a08dd9" />
深度研究不是简单的单次问答，而是模拟人类研究员的工作方式——自动拆解问题、多轮搜索、逐步推理、综合归纳。核心链路大致是：
1. 问题拆解：将用户输入的研究课题自动分解为若干子问题
2. 自主搜索：循环调用搜索引擎，对每个子问题独立检索，可执行数十次搜索
3. 内容阅读与筛选：爬取并精读网页全文，而非只看摘要片段
4. 迭代推理：根据中间结果判断是否需要补充检索，动态调整研究路径
5. 报告生成：将所有信息整合，输出结构化长篇报告，并附带引用来源
底层依赖 Kimi 超长上下文窗口能力（支持百万 token），使其能同时记住大量检索到的材料进行综合。

2.怎么用深度研究
入口很简单，在 Kimi 对话框中：
- 点击输入框旁的「深度研究」按钮，或在问题前选择深度研究模式
- 输入你的研究问题，点击发送
- Kimi 会先展示研究计划（列出它打算探索的方向），用户可以确认或修改
- 然后进入自动执行阶段，过程中可以看到它实时搜索了哪些关键词、访问了哪些网页
- 最终输出一份带有目录结构、多章节的深度报告，每个论点都有来源标注

你只需输入你的研究主题和具体要求，Kimi 会立即启动自主研究流程。整个过程通常需要 10～25分钟，取决于问题复杂度。
Kimi会给你一份深度长文和一个交互网页。研究完成后，你会收到一条通知。普通问答是"一问一答"，深度研究是"委托一个任务"——它会自主规划、多轮执行、最终交付，更接近让一个助理帮你做了几小时的资料整理工作。
适合信息密度高、需要多来源交叉验证的复杂问题，而不太适合简单的即时性问题。

4.深度研究的边界
目前深度研究在算命、预测彩票等场景回答并不准确。
如果你在第一轮对话中输入内容超长，它可能抓不住重点。
另外，涉及到国家安全、敏感内容它可能会拒绝回答。

5.怎么使深度研究内容更准确？
1.给问题加约束条件。具体可以在 prompt 里加：
- 时间范围：限定研究时间窗口，避免过时信息混入
- 地域范围：国内/海外/某个具体市场
- 信息来源偏好：如"优先引用官方报告、学术论文、头部媒体"，医学领域只检索PubMed网页相关文献信息，法学领域只检索北大法宝相关内容信息。
你也可以使用这个万能提示词给检索加一些约束：

请对以下问题进行深度研究：[你的问题]
研究要求：
1. 每一个数据、结论、事实性表述，必须标注具体来源（网站名+标题，最好有发布时间）
2. 如果某个信息你没有检索到可靠来源，请明确说"未找到可靠来源"，不要推测或补全
3. 区分三类内容并用括号标注：
   - [已证实]：有明确来源支撑
   - [多源一致]：多个独立来源说法相同
   - [存疑]：来源有限或说法不一，需用户自行核实
4. 不同来源之间如有矛盾，请并列呈现，不要自行裁决哪个正确
5. 禁止用"据悉""有报道称"等模糊表达替代具体来源

2.在意图澄清部分补充内容。深度研究会先给你提出相关问题，这是最重要的干预时机。你可以在这里：
- 删掉它走偏的子问题
- 补充它遗漏的关键维度
- 指出某个方向的优先级更高
这一步改得好，后续整个报告的方向就不会跑偏。

3.在Kimi完成深度研究之后，进行进一步追问。比如：
- 报告里提到 XX 数据，能告诉我原始来源是哪里吗？
- 这个结论有没有反面观点或不同数据？
- 第三部分的判断，有没有更新的信息支撑？
这样可以找到检索的信息盲区，也能验证某个关键结论是否稳固。

6.深度研究使用案例
1. 你需要搬救兵查资料
不是简单问答，而是需要整合分散信息、交叉验证数据的任务。比如对标竞品，看看功能差异背后的技术路径选择。梳理政策脉络、市场规模、头部玩家动态。为论文找理论支撑，或快速摸清某个研究方向的现状与空白。

示例提示词：我是一名专业的证券公司分析师，请帮我从金融专业角度，深度分析泡泡玛特这家公司。分析内容包括但不限于： 1.公司概况：业务模式、护城河及行业地位；2.行业概况、上下游情况、未来发展方向； 3.财务数据：营收及利润变化趋势影响因素、资产负债与现金流等； 3.合法合规经营情况，是否收到相关处罚、具体处罚事由和风险；4.技术分析：价格趋势、指标及支撑/阻力位；5.市场情绪：评级、舆情与新闻影响； 6.竞品对比：主要竞争对手的对比、各自市场份额和财务指标； 7.估值与健康：PE/PB/DCF估值、合理性；8.主要风险：行业竞争风险、政策风险、地缘风险等 。

2. 问题有多种思路
单维度问题用普通对话就好，但以下情况建议交给深度研究。比如多维度对比，了解DeepSeek 和 Kimi 的技术架构差异。跨领域分析，了解跨境电商在东南亚的合规风险。或者你只想做一次旅行规划？但是有很多纠结因素，欢迎来问问Kimi。

示例提示词：我想去新疆的北疆自驾游，我帮出一个五天四夜的攻略，包括吃喝玩乐。我们家一共四个人，我和我妻子，还有一个男孩和一个女孩，男孩六岁，女孩四岁。小朋友们比较活泼，喜欢走路，但是特别危险的路线不要推荐。

4. 结论需要经得起追问
当你要把结果贴进 PPT、发给老板、或写进报告时，深度研究会自动标注信源，每个数据点都可追溯。并且生成双份成果，详细长文供你深度阅读，可视化网页直接用于演示。如果只是闲聊或简单任务，请保持关闭。

示例提示词：我是一名传媒专业研究生，正在进行"数字鸿沟"的学术研究。请帮我进行系统性深度调研。我希望你能够提供至少15篇中英文核心期刊文献。给我这个概念的定义，近3-5年的文献内容，并给我3个可供深入研究的细分选题建议。重点参考2020年以来的最新研究与政策文件，特别关注2023年AI爆发后的新研究动态。

##Agent功能##

1.Agent是什么
Kimi Agent 是一套自主任务执行系统，它以Kimi K2.5为核心大脑，能够自主理解用户的自然语言目标。Agent通过规划、调用外部工具、多步骤推理，独立完成从信息收集到最终输出的完整工作流，整个过程不需要用户的干预。用户只说"要什么"，Agent 自己决定"怎么做"，并最终交付结果。
Kimi 的 Agent 功能最早内测名为 OK Computer，后改名为Agent。OK Computer这个名字致敬了英国摇滚乐队 Radiohead 1997 年发行的同名专辑《OK Computer》，这张专辑的核心主题是"技术时代的人类异化"。Kimi 用这个名字暗示：AI 与电脑的关系，正在进入一个新阶段。
<img width="2996" height="1494" alt="image" src="https://github.com/user-attachments/assets/7552e07b-6255-4692-8d7f-337d9424a42b" />

在实际使用中，K2.5 能作为单一 Agent 运行，支持工具增强的 Agent 模式，可以规划、调用工具、处理多步骤工作流，产出结构化内容如表格、报告、计划和代码。

2.Agent Swarm 模式
Kimi K2.5 通过 PARL（Parallel-Agent Reinforcement Learning）训练方法，学会自主指挥一个最多包含 100 个子 Agent 的群体，执行最多 1,500 步的并行工作流，且无需预定义角色或手工设计工作流程。与单 Agent 方案相比，Agent Swarm 在大规模搜索场景中将完成目标所需的最少关键步骤减少了 3× 至 4.5×，折算为实际执行时间最高缩短 4.5 倍。

3.怎么用Agent模式
在 Kimi 对话框中：
- 点击输入框旁的「Agent」按钮，或在问题前选择 Agent 模式
- 用自然语言描述你想完成的任务目标，点击发送
- Kimi 会先展示执行计划（列出它打算拆解的子任务和步骤），用户可以确认或修改
- 然后进入自动执行阶段，过程中可以看到它实时调用了哪些工具、搜索了哪些内容、生成了哪些中间文件
- 最终交付一份可直接使用的结构化成果，包括文档、表格、PPT、网站或代码，支持预览、下载和分享
这里有一些提示词供你参考：
暂时无法在飞书文档外展示此内容
你只需输入任务目标和具体要求，Kimi 会立即启动自主规划与执行流程。整个过程通常需要 5～20 分钟，取决于任务复杂度。Agent 完成后，你会收到一条通知，并获得可直接交付的成果文件。

4.Agent的边界
目前Agent生成的网页内容只有前端界面，因为没有后端数据库，所以无法部署或上传数据内容。
Agent运行在云端，无法直接打开你电脑C盘的文件，也不能登录你的企业内网系统。你需要把文件上传到对话中，处理完再下载回去。
如果你需要让Agent连续处理几十份文件时，它可能会”遗忘”最初的部分细节。你可以把大任务拆成2-3个阶段，分批提交给Kimi。

5.Agent使用案例
1.你的任务需要多步操作
你的任务不是简单问答，而是需要一连串动作才能完成。如果你需要分析完数据后生成图表，再整理成报告。或是读取合同后提取关键条款，再对比历史版本找出差异。只要你的需求里出现了然后、再、最后这样的连接词，就是Agent的专长领域。
这是用户在看剧时想整理一下剧情所在的历史时间线，Kimi会给你一份Excel梳理的历史时间轴，还有可视化报告。

示例提示词：我最近在看《太平年》，这个电视剧很火。这个剧主要讲的是五代十国的故事，吴越国君主钱弘俶于北宋初年决定纳土归宋。请帮我准备一份 PDF 格式的报告，详细说明那个时代到底发生了什么，以及涉及的具体事件。 另外，请提供一个 Excel 表格，列出所有关键信息及时间点。 我对这段历史一点也不懂，但最近到处都能看到这个剧的新闻，请给我提供完整的介绍。


2.你要进行原材料加工
你不只是让Kimi看看这份文件，而是需要把它从一种形态转化为另一种可用形态。比如把杂乱的Excel变成规范的数据看板，把PDF合同里的文字提取成可编辑的表格，或是把一张图片转化成一个网页。只要涉及“读取→处理→输出新文件”的完整流程，就是Agent发挥优势的时候。
这是用户看到了一张好看的图片想生成一个网页，Kimi会给你网页预览和所有网页源代码。

示例提示词：参考这个网页风格，帮我生成一个网页，可以交互。

3. 优化每周都要做的重复工作
每周都要翻译外文资料、整理参考文献格式、梳理研究领域的文献脉络——这些学术研究里重复性高、有固定流程的体力活，Agent可以记住你的处理习惯，建立自动化工作流，下次遇到同类任务直接套用，把你从机械劳动中解放出来。
这是用户在写文献时遇到的场景，Kimi会给你研究发展时间线图示和文献综述文档。

示例提示词：给你一个PDF文件，帮我翻译这篇文章，给我中英文对照内容，并且把参考文献中提到的文章帮我梳理逻辑图示和word文档，方便我写一篇这个方向的文献综述。

如果只是闲聊或一次性简单问答，保持普通对话即可。一旦涉及多步骤操作、文件深加工、重复性流程，请果断开启Agent，你说得越具体，Kimi执行得越精准。

Kimi PPT
1.Kimi PPT是什么
Kimi PPT 是 Kimi 推出的自动化PPT生成工具。K2.5 支持格式转换，可将 3 万字长文一键转换为 PPT，自动优化排版与设计。 模型先对输入材料做语义理解，提炼核心论点，自动规划章节结构与逻辑顺序，再生成每页的标题、正文、配图建议。PPT工具会根据内容类型选择最合适的视觉叙事方式，数据用图表、流程用箭头、对比用并列结构。目前PPT输入支持上传 PDF、Word、Excel、TXT、图片格式，上传材料后，Kimi 会基于内容自动生成大纲。
2.怎么用PPT功能
目前使用PPT工具有两个入口：
① PPT 功能：轻量级入口，适合快速从主题或文档生成 PPT，聚焦内容生成 + 模板套用。
② Agent 模式下的 PPT 生成：选择Agent模式，在提示词中要求Kimi生成PPT。
使用步骤如下：
首先，在对话框里写 PPT 主题和要求，比如"我想要2026年AI发展趋势研究报告，需要商务风格，给我10页左右PPT"。如果你有需要Kimi参考的Word或者PDF文档，可以直接拖动上传到对话框，Kimi 会根据内容帮你列大纲。
<img width="1280" height="757" alt="image" src="https://github.com/user-attachments/assets/ed5ec5dc-6bab-4ed4-b4f7-5c9e40ca58c5" />
之后，你可以选择智能布局或者经典模板。智能布局会根据PPT内容自动调整PPT页面布局，生成效果较好。选择完成后点击发送，等待系统完成后下载PPT格式文件。
<img width="1280" height="786" alt="image" src="https://github.com/user-attachments/assets/4e99edf0-6ac6-4e51-893e-fdd324dcc325" />

3.适用人群
职场人士： 快速完成汇报、方案、周报转 PPT，从"做 PPT"解放出来专注内容本身。
学生/研究者： 将论文、综述、调研报告直接转化为答辩或演讲用 PPT。
创业者： 快速生成融资路演PPT，节省与设计师的沟通成本。
Kimi 的目标是让 Agent 逐渐承担起 Office 软件中那些枯燥的技术性工作，让用户从格式的繁琐中抽身，将全部心力投入到更有价值和创造性的工作中。

Kimi 网站
1.Kimi 网站是什么
Kimi 的网站生成功能是 K2.5 发布后最受开发者关注的核心能力之一。Kimi的网站功能不仅能看懂静态 UI 布局，还能理解动态交互逻辑，比如滚动触发特效、复杂的卡片翻转动画等。在它接到指令后，能够直接生成包含完整 CSS 和 JS 逻辑的可用代码。

Kimi 文档
1.Kimi文档是什么
Kimi 文档有较强的文本适应能力，它能够系统理解 Office 的运作方式，包括表格里的单元格关系、Word 的标题层级、PDF 的排版逻辑。Kimi 能像专业律师一样审阅复杂合同条款，并自动插入修订批注。 具体能力包括：
- 从零生成长篇专业文档，包括白皮书、研究报告、合同、方案等格式
- 在原有 Word 文档里插入批注、修改意见，保留原格式
- 标题层级、正文间距、引用和列表格式处理干净
- 把 3 万字论文变成一份精准摘要文档，直接交付准专业水平的办公文档

Kimi 表格
1.Kimi 表格是什么
Kimi表格能够系统理解 Office的运作方式，包括表格里的单元格关系、多 Sheet 之间的联动逻辑，最终在 Web 端直接产出文件。Kimi在做复杂表格时，会尽量把数字和文字格式都保留对。还能主动用函数把同一个文件里的多张表连起来用，节约用户做表格的时间。
2.怎么用Kimi 表格
首先点击对话框底部的「Agent」按钮，或在侧边栏选择「表格」快捷入口。
