会员权益说明

Kimi 会员是一项灵活的订阅计划，我们提供包月与包年两种订阅方式，您可以根据使用需求选择最合适的订阅计划。

Adagio 为免费版用户权益，会员权益（按周期计费，自动续订）分为 4 档：
Andante，入门版用户权益；
Moderato，基础版用户权益；
Allegretto，高级版用户权益；
Allegro，进阶版用户权益。

Adagio（免费）权益介绍

权益内容
免费获得 3 次 Agent 的使用额度；
免费获得 1 次深度研究的使用额度；
免费获得 3 次 PPT 的使用额度；

可正常通过网页、APP 等端口访问并使用 Kimi 的部分模型（以实际提供为准）；
支持联网搜索能力。

关于 Agent 使用额度： Kimi 中的 Agent、网站、文档、表格等功能（不包括深度研究、PPT功能），统一归属于 Agent 功能。您的 Agent 使用额度可在这些功能之间共享使用，即您使用上述任一功能均会消耗 Agent 使用额度。不同功能因任务复杂度不同，可能消耗的额度有所差异，具体以实际使用时的页面提示为准。

关于 Kimi Claw： Kimi Claw 与 Kimi Code 共享可用额度，按照会员档位下发对应额度。如会员过期，Kimi Claw 云资源将为您保留七天，之后会永久关停并释放资源，届时您的相关数据和资源都将清空，请您务必注意。不支持同时运行两个及以上的 Kimi Claw 任务。

API 使用不属于会员权益范围，费用需单独支付，如有需求请访问我们的开放平台。

除试用期会员权益外，月包和年包的会员权益在订阅期间均将以月为周期发放，会员权益有效期自订阅生效之日起算，有效期为一个月，有效期内未使用的会员权益将在到期后自动失效。到期未使用的会员权益，不予退款、结转、退换。

使用限制

为了确保所有用户都能获得流畅的体验，Kimi 会员可能会设置使用限制，例如消息上限，尤其是在高需求时段。这些限制可能会根据系统状况而有所不同。
